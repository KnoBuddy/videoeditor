# Features

## GUI
### Video Preview
### Crop any portion of the video
### Increase or decrease volume
### Adjust resolution or bitrate
### Status Bar
### .mp4 files supported

## Instructions for Windows
1. Unzip .zip
2. Run exe

## Instructions for Linux
1. Unzip .tar.gz
2. ./VideoEditor
